## Run locally
1) npm i
2) npm run dev
3) Open http://localhost:3000

## Deploy to Vercel
1) npm i -g vercel
2) vercel login
3) vercel
4) vercel --prod

## Domain
Add alexhardinan.com in Vercel dashboard.
Update DNS records to Vercel nameservers.

## Email
Contact form uses mailto fallback.
For real sending, connect Resend or SMTP later.
